# FitCheck ⚡
> **Workout. Notebook. Reinvented.**

FitCheck is a high-performance fitness suite built with Flutter, designed for a "stealth-mode" aesthetic. It moves away from bloated fitness apps, offering a clean, dark UI with vibrant blue accents, smooth transitions, and a professional-grade workout tracking experience.

---

## 🌌 The Vibe
* **Stealth Blue Aesthetic:** A permanent, high-contrast dark mode.
* **Premium Animations:** Custom cross-fade page routing and entry transitions.
* **Celebratory Logic:** Integrated confetti system for workout completion.

---

## 🚀 Key Features

### 1. Daily Dashboard
Track your grind at a glance. Features a **Syncfusion Radial Gauge** to visualize daily goals and interactive trackers for:
* 💧 **Water Intake**
* 🏃 **Step Counter**
* 🔥 **Calories Burned**

### 2. Active Workout Hub
Inspired by the "Strong" app layout, this is a real-time workout notebook:
* ⏱️ **Functional Stopwatch:** Accurate to the second.
* 📊 **Set Tracking:** Pro-style table for Sets, KG, and Reps.
* 💡 **Pro-Tips:** Contextual help like "Watch back rounding" during lifts.
* 🎉 **Finish Logic:** Explosive confetti and summary dialog upon completion.

### 3. Body Analysis (BMI) diagnostic
A dedicated calculator with haptic feedback:
* **Dual System:** Support for both Metric (cm/kg) and Imperial (in/lb) units.
* **Visual Feedback:** Needle pointers and color-coded health ranges.

### 4. Workout Squad (API Integration)
Real-time community features:
* **Live Fetching:** Pulls data from external REST APIs.
* **Local State Management:** Add or remove workout buddies with instant UI updates.
* **Search Engine:** Filter partners by name in real-time.

### 5. Personal Profile
* **Image Picking:** Upload a custom JPEG from your gallery or use the default **Ramadan Guts** PFP.
* **Achievement Badges:** Horizontal scroll of streaks and trophies.
* **Settings:** Manage notifications and fitness goals (Weight Loss / Muscle Gain).

---

## 🛠️ Tech Stack
* **Framework:** Flutter (Material 3)
* **Gauges:** `syncfusion_flutter_gauges`
* **Animations:** `confetti`, `AnimatedSwitcher`, `PageRouteBuilder`
* **Utilities:** `image_picker`, `dart:async`, `http`
* **Storage/Logic:** Haptic feedback, local list mutations, stateful transitions.
